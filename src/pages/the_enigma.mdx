# Historical Cryptography: The Enigma Machine

<div class="enigma-hero-card">
  <div class="enigma-hero-text">During World War II, Nazi Germany deployed a cipher device so sophisticated that the Allies believed it to be <strong>unbreakable</strong>. This machine — the <strong>Enigma</strong> — could produce roughly 159 quintillion possible configurations, making brute-force decryption seemingly impossible.</div>
  <div class="enigma-hero-subtext">The breaking of the Enigma code — first by Polish mathematicians under Marian Rejewski in the 1930s, and later by Alan Turing's team at Bletchley Park — is widely credited with shortening the war by at least two years.</div>
</div>

## How the Enigma Works

<div class="enigma-how-works">Understanding the Enigma requires grasping three core components that work together to create a <em>polyalphabetic substitution cipher</em> — one where the same input letter can encode to different outputs depending on the machine's internal state.</div>

<div class="enigma-components-grid">
  <div class="enigma-component-card">
    <span class="enigma-component-icon">R</span>
    <h4>The Rotor</h4>
    <p>A scrambled wiring mapping (permutation of 26 letters) that shifts position after each keypress, changing the entire cipher.</p>
  </div>
  <div class="enigma-component-card">
    <span class="enigma-component-icon">T</span>
    <h4>The Reflector</h4>
    <p>A fixed wiring of paired letter swaps. No letter maps to itself — this property was the Enigma's crucial weakness.</p>
  </div>
  <div class="enigma-component-card">
    <span class="enigma-component-icon">S</span>
    <h4>The State</h4>
    <p>An integer (0–25) representing the rotor's current position. It advances with each keypress, making every encoding unique.</p>
  </div>
</div>

### Signal Path

The signal passes through the machine in a specific order: **Input → Rotor (forward) → Reflector → Rotor (reverse) → Output**. Use the interactive diagram below to trace the encoding of any letter at any rotor state.
